# anpt.dev

A technical blog exploring distributed systems, database internals, and the craft of building reliable software at scale.

## Run Locally

**Prerequisites:** [Bun](https://bun.sh)

1. Install dependencies:
   ```bash
   bun install
   ```
2. Copy `.env.example` to `.env` and fill in your credentials
3. Run the dev server:
   ```bash
   bun run dev
   ```

## Build

```bash
bun run build
```
